# Tiexo-wormhole

## Install

```bash
npm install
```

## Develop testnet

```bash
REACT_APP_CLUSTER=testnet npm start 
```

## Develop mainnet

```bash
REACT_APP_CLUSTER=mainnet npm start 
```

## Build 

```bash
REACT_APP_CLUSTER=mainnet npm run build
```

## Start 

```bash
REACT_APP_CLUSTER=mainnet npm run start
```
